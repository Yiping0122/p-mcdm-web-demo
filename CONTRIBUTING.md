# 贡献指南

感谢您对 P-MCDM 项目的关注！

## 重要声明

本项目是一个**研究原型**，用于展示建筑材料多准则决策分析的界面设计。

### 当前状态

- ✅ **可贡献**: UI组件优化、Bug修复、性能改进、文档完善
- ❌ **暂不接受**: 新增核心算法、修改mock数据结构（论文发表前）

## 如何贡献

### 报告问题

如果您发现界面显示问题或交互Bug，请提交Issue：

1. 描述问题现象
2. 提供浏览器和操作系统信息
3. 如有必要，附上截图

### 提交代码

1. Fork 本仓库
2. 创建新分支：`git checkout -b fix/xxx` 或 `feature/xxx`
3. 提交更改：`git commit -m "描述"`
4. 推送分支：`git push origin xxx`
5. 创建 Pull Request

### 代码规范

- 使用 TypeScript
- 遵循现有代码风格
- 保持组件原子化设计

## 许可限制

由于本项目采用 **CC BY-NC-ND 4.0** 许可证：
- 您不能将此代码用于商业目的
- 您不能修改后重新分发
- 但您可以Fork并提交PR到原仓库

## 联系方式

如有疑问，请通过Issue或邮件联系。
