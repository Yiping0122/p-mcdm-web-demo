# 截图目录

请在本地运行项目后，截取以下界面图片并保存到此目录：

## 需要的截图

1. **screenshot-component.png** - 组件结果页面
   - 显示某个组件（如External Wall）的材料排名
   - 包含箱线图和统计表格

2. **screenshot-building.png** - 建筑组合页面
   - 显示建筑组合评估结果
   - 包含帕累托前沿图

3. **screenshot-analysis.png** - 方法分析页面
   - 显示方法一致性分析
   - 包含Spearman相关系数

4. **screenshot-material.png** - 材料查询页面
   - 显示材料查找功能
   - 包含指标分布图

## 截图建议

- 尺寸：推荐 1280×720 或 1920×1080
- 格式：PNG（保留透明度）
- 命名：必须与上述文件名完全一致

截图完成后，README中的预览图片将自动显示。
